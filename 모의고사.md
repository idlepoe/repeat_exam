모의고사 기능

ExamTypeListPage 에서 '모의고사' 소타이틀에 제빵기능사, 제과기능사를 선택하여 진입할 것.

모의고사가 시작될 경우 문제, 시작시간을 storage에 저장할 것.

assets/json/exams
제빵 기능사 bread_ 
제과 기능사 pastry_

type에 맞는 파일 중에

제조이론 30문제, 재료과학 15문제, 식품위생학 10문제, 영양학 5문제 구성할 것. 

subject = '제조이론' | '재료과학' | '식품위생학' | '영양학'


문제는 1개씩 표시

시간 60분 지날 경우 알림(다이얼로그 표시. 남은시간이 0초가 된 타이밍에 1번)만 하고 -분-초으로 하여 계속 표시할 것. 타이머는 신경 쓰지 않아도 됨. 문제 시작시간으로부터 계산 표시 (매 타이머는 시작시간으로부터 60분을 남은 시간으로 표시시)

각 타이머는 각 문제 화면에서 문제 시작시간으로 계산하여 표시

1.appbar
뒤로 가기(버튼), 58분(남은시간)5초/60분, 시험종료(버튼), 답안확인(버튼)

뒤로가기: get.back()
남은시간은 시작시간을 참고하여 현재시간으로 하여 실시간 남은시간을 표시할 것.

58분(남은시간)/60분: (시작시간으로부터 60분 남은 시험시간 표시, 문제별 타이머는 표시할 필요 없음)

시험종료: '모의고사를 종료하시겠습니까?' '확인' 확인 다이얼로그 표시 후 확인 후 처리
문제, 답안, 시작시간 storage를 리셋할 것. 모의고사 데이터만 지울것. history에 남기지 않음. get back

정답확인: 다이얼로그 표시

번호, 문제, 답안으로 하여 vertical list가 표시될 것.
선택시 해당 문제로 이동할 것.
미응답 행 표시 방법(빈 칸).
문제는 1줄 overflow 표시시

2.본문
문제와 보기가 표시되고, 정답을 선택할 경우 정답을 storage에 저장할 것.
정답 선택시 다음 문제로 이동할 것

문제의 이미 선택한 정답이 있을 경우 정답은 하이라이트 표시할 것.

3.하단 (이전 기출문제 하단과 동일)
이전, 변경, 다음

이전: 이전 문제로 이동
다음: 다음 문제로 이동

다음 문제가 존재하지 않을 경우 전체 문제의 답안이 작성되었는지 확인하고 '풀지 않은 문제가 있습니다. 이동하시겠습니까?' 알림을 표시하고 '해당 문제로 이동' 버튼을 표시 해당 버튼을 누를 경우 답안이 작성되지 않은 문제로 이동

다음 문제가 존재하지 않을 경우 전체 문제의 답안이 작성되어 있는 경우 모의고사 문제, 답안, 시작시간, 종료시간을 모의고사  history키로 별도 storage에 저장할 것. 모의 고사 데이터는 전부 지울 것.
